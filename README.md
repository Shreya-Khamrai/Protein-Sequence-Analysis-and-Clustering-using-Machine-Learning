# Protein-Sequence-Analysis-and-Clustering-using-Machine-Learning

I used Python, Biopython, Pandas, NumPy, Scikit-learn, Matplotlib, and Plotly to analyze 60,000 protein sequences from five functional protein classes. I developed a feature engineering pipeline to calculate Molecular Weight, Isoelectric Point, Instability Index, Aliphatic Index, Aromaticity, GRAVY, Flexibility, Charge Density, Basic–Acidic Ratio, Hydrophobic–Polar Ratio, and amino acid composition. I performed data preprocessing, correlation analysis, feature selection, standardization, and Principal Component Analysis (PCA) for dimensionality reduction. I implemented K-Means, Gaussian Mixture Model (GMM), and Hierarchical Clustering, and evaluated clustering performance using the Silhouette Score. I also performed class-wise clustering for Enzyme, Receptor, Transport, Structural, and Other proteins, and generated PCA plots, correlation heatmaps, cluster summaries, and statistical analyses to interpret biochemical patterns.
